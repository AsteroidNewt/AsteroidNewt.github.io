Estructura y funcion

La Estructura y Función Computacional abordan la organización interna y el funcionamiento de las computadoras.

  1. Hardware y software

    2. La computadora se compone de hardware (componentes físicos) y software (programas y datos).


  2. Memoria

    3. Almacena datos e instrucciones temporalmente. Incluye RAM (memoria de acceso aleatorio) y almacenamiento secundario.


  3. Entrada/Salida (E/S)

    4. Permite la interacción con dispositivos externos, como teclados, monitores y unidades de almacenamiento.


  4. Sistema de Coordenadas

    5. La CPU coordina las operaciones mediante buses de datos y control.


  5. Ciclo de Instrucción.

    6. Proceso repetitivo que incluye la búsqueda, decodificación y ejecución de instrucciones.


  6. Estructuras de Datos.

    7. Organización de datos para facilitar el acceso y la manipulación eficiente.


  7. Algoritmos.

    8. Conjunto de pasos definidos para realizar una tarea específica. Fundamental para el software.


  8. Redes y Comunicación

    9. La conexión de computadoras permite la comunicación y el intercambio de datos.


  9. Sistemas Operativos.

    10. Software que gestiona recursos y facilita la interacción entre hardware y software.


  10. Seguridad Informática

    11. Medidas para proteger sistemas y datos contra accesos no autorizados.


  11. Arquitecturas Paralelas y Distribuidas

    12. Uso de múltiples procesadores o la conexión de varias computadoras para mejorar el rendimiento.
